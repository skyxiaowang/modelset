# Security Policy

## Reporting a Vulnerability

We take the security of ModelSet and our users seriously.

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT** open a public GitHub issue
2. Contact us via our [Help Center](https://modelset.top/about)
3. Include detailed steps to reproduce the vulnerability
4. Include the potential impact

## What to Expect

- We will acknowledge your report within 48 hours
- We will investigate and provide a timeline for fixing
- We will credit responsible reporters in our changelog

## Scope

- API authentication and authorization
- Data exposure or leakage
- Account takeover vectors
- Infrastructure vulnerabilities

## Out of Scope

- Social engineering attacks
- Physical security issues
- DOS/DDOS attacks

Thank you for helping keep ModelSet secure!
