## Quick Start

### Step 1: Get your API Key

Visit [ai.modelset.top/login](https://ai.modelset.top/login) to register and get your API Key.

```bash

export ANTHROPIC_AUTH_TOKEN="sk-xxxx"
```

### Step 3: Start using

```bash

```
